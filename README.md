🎮 Busca de Pokémon - PokéAPI

Projeto desenvolvido utilizando HTML, CSS e JavaScript para consumir a API pública PokéAPI e buscar informações de um Pokémon pelo número informado.

---

🚀 Funcionalidades

- Campo para digitar o número do Pokémon
- Botão para realizar a busca
- Exibição de:
  - Nome do Pokémon
  - Imagem
  - Dois ataques
- Validação de número inválido
- Tratamento de erro caso o Pokémon não exista

---

🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Fetch API
- API pública PokéAPI

---

🔗 API utilizada

PokéAPI  
https://pokeapi.co/

Endpoint base:
https://pokeapi.co/api/v2/pokemon/

Exemplo:
https://pokeapi.co/api/v2/pokemon/1

▶️ Como executar o projeto

1. Abra a pasta do projeto no VS Code
2. Instale a extensão **Live Server** (caso ainda não tenha)
3. Clique com o botão direito no arquivo `index.html`
4. Selecione **Open with Live Server**

O projeto será aberto no navegador utilizando um servidor local.
